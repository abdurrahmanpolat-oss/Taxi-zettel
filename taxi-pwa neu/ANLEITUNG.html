🚕 Taxi Tageszettel — Installation

So installierst du die App auf deinem Handy

📁 Schritt 1 — Dateien hochladen (einmalig)

1
Gehe zu github.com und erstelle ein kostenloses Konto (falls noch nicht vorhanden)
2
Klicke oben rechts auf + → New repository
3
Name: taxi-zettel → Public auswählen → Create repository
4
Klicke auf uploading an existing file und lade alle Dateien aus dem ZIP hoch:

index.html · manifest.json · sw.js · Ordner icons/
5
Klicke auf Commit changes

🌐 Schritt 2 — GitHub Pages aktivieren

1
Im Repository oben auf Settings klicken
2
Links auf Pages klicken
3
Unter Branch: main auswählen → Save
4
Nach 1–2 Minuten erscheint deine App-Adresse:
https://DEINNAME.github.io/taxi-zettel

📱 Schritt 3 — App auf Handy installieren

🍎 iPhone (Safari):

1
Öffne die App-Adresse in Safari
2
Tippe unten auf das Teilen-Symbol (Viereck mit Pfeil nach oben)
3
Wähle „Zum Home-Bildschirm"
4
Tippe oben rechts auf Hinzufügen — fertig! 🎉

🤖 Android (Chrome):

1
Öffne die App-Adresse in Chrome
2
Tippe oben rechts auf die 3 Punkte (⋮)
3
Wähle „Zum Startbildschirm hinzufügen" oder „App installieren"
4
Tippe auf Hinzufügen — fertig! 🎉
💡 Wichtig: Die App funktioniert auch offline — sobald sie einmal geladen wurde. Alle Daten werden auf dem Handy gespeichert.
